# Research Agent (MULTI-AGENT SYSTEMS: Homework 4 - Custom ReAct)

Цей проєкт реалізує автономного ШІ-агента (Research Agent), який здатний отримувати запитання від користувача, самостійно шукати інформацію в інтернеті, читати статті та генерувати структуровані звіти у форматі Markdown. 

**Відмінність цієї версії:** Повністю власна реалізація ReAct (Reasoning and Acting) циклу без використання абстракцій LangGraph. Агент напряму керує пам'яттю, обробкою інструментів через JSON Schema та логікою циклу.

## 🏗 Архітектура проєкту

* **LLM Провайдер:** За замовчуванням використовується **Google Gemini 3.1 Flash lite** (через LangChain для базового доступу до API).
* **Власний ReAct Loop:** Агент самостійно ітерується в циклі `while`, аналізує потребу у виклику інструментів (Tool Calling), виконує Python-функції та повертає їх результати назад в контекст LLM.
* **Tool Calling API:** Інструменти передаються в модель як чисті структури `JSON Schema`, що робить механізм абсолютно прозорим.
* **Context Engineering:** Реалізовано механізм обрізання занадто великих текстів з веб-сторінок (до 8000 символів), щоб не переповнювати контекстне вікно моделі.

### Доступні інструменти
1. `get_current_datetime`: Отримання точного поточного часу для розуміння контексту.
2. `web_search(query: str)`: Пошук в інтернеті за допомогою DuckDuckGo (`ddgs`). 
3. `read_url(url: str)`: Витягування повного корисного тексту зі сторінки за допомогою `trafilatura`.
4. `write_report(filename: str, content: str)`: Збереження фінального звіту від агента у вигляді `.md` файлу.
