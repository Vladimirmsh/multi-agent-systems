# Research Agent (MULTI-AGENT SYSTEMS: Homework 5 - Custom ReAct + RAG)

Цей проєкт реалізує автономного ШІ-агента (Research Agent), який здатний самостійно шукати інформацію як в інтернеті, так і в **локальній базі знань (RAG-система)**.

**Відмінність цієї версії:** Додано гібридний пошук (Semantic Search + BM25) по векторній базі `Qdrant` із використанням Cross-Encoder reranking (`BAAI/bge-reranker-base`) та Google Embeddings (`gemini-embedding-001`).

## 🏗 Архітектура проєкту

* **LLM Провайдер:** За замовчуванням використовується **Google Gemini 3.1 Flash lite**.
* **Власний ReAct Loop:** Агент самостійно ітерується в циклі `while`.
* **Vector DB & RAG:** Векторна БД `Qdrant`, ембеддінги Google, пайплайн обробки документів PDF.
* **Hybrid Search:** `EnsembleRetriever` для об'єднання лексичного та семантичного пошуку.

### Доступні інструменти
1. `get_current_datetime`: Отримання точного поточного часу.
2. `web_search(query: str)`: Пошук в інтернеті. 
3. `read_url(url: str)`: Витягування повного корисного тексту зі сторінки.
4. `knowledge_search(query: str)`: Пошук в локальній базі знань (PDF документи).
5. `write_report(filename: str, content: str)`: Збереження фінального звіту.
