# Research Agent (MULTI-AGENT SYSTEMS: Homework 3)

Цей проєкт реалізує автономного ШІ-агента (Research Agent), який здатний отримувати запитання від користувача, самостійно шукати інформацію в інтернеті, читати статті та генерувати структуровані звіти у форматі Markdown.

## 🏗 Архітектура проєкту

Агент побудований за патерном **ReAct (Reasoning and Acting)** з використанням фреймворку `langgraph`. 

* **LLM Провайдер:** За замовчуванням використовується **Google Gemini 3.1 Flash lite** (через `ChatGoogleGenerativeAI`). Архітектура дозволяє змінити провайдера (наприклад, на OpenAI) зміною одного рядка в `agent.py`.
* **Пам'ять (Memory):** Використовується `MemorySaver` (checkpointer) для підтримки контексту діалогу в межах сесії. Агент пам'ятає попередні повідомлення та результати пошуку.
* **Context Engineering:** Реалізовано механізм обрізання занадто великих текстів з веб-сторінок (до 8000 символів), щоб не переповнювати контекстне вікно моделі.

### Доступні інструменти (Tools)
1. `web_search(query: str)`: Пошук в інтернеті за допомогою DuckDuckGo (`ddgs`). Повертає список релевантних посилань із короткими сніпетами.
2. `read_url(url: str)`: Витягування повного корисного тексту зі сторінки за допомогою `trafilatura` (ігноруючи рекламу та меню).
3. `write_report(filename: str, content: str)`: Збереження фінального звіту від агента у вигляді `.md` файлу в директорію `output/`.

## ⚙️ Вимоги та встановлення

* `langchain >= 0.3.0`
* `langgraph >= 0.2.0`
* `ddgs >= 7.0`
* `trafilatura >= 2.0.0`
