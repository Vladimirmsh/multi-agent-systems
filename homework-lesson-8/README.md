# Multi-Agent Research System (Homework 8)

Цей проєкт реалізує мультиагентну систему на базі LangGraph із застосуванням **Supervisor Agent**, який координує трьох спеціалізованих суб-агентів:

1. **Planner Agent:** Декомпозує запит у структурований план (`ResearchPlan`).
2. **Researcher Agent:** Виконує дослідження за планом, користуючись вебом та локальною базою знань.
3. **Critic Agent:** Здійснює незалежну перевірку фактів, оцінює повноту та актуальність даних, повертає структурований вивід (`CritiqueResult`). 
4. **Supervisor Agent:** Оркеструє цикл (Plan → Research → Critique) та робить запит до користувача перед збереженням файлу.

## Особливості
- **HITL (Human-In-The-Loop):** Збереження звіту `save_report` вимагає підтвердження користувача (Approve / Edit / Reject).
- **Структуровані виводи (Pydantic):** `Planner` та `Critic` повертають Pydantic-об'єкти (Structured Outputs).
- **Вбудована пам'ять:** Для підтримки сесії під час зупинок HITL використовується `InMemorySaver`.
